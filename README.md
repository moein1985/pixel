# Pixel — شناسنامه هوشمند کشاورزی

پلتفرم ملی B2B/B2G برای تحول دیجیتال در حوزه کشاورزی ایران.

## Tech Stack

- **Frontend:** Next.js + TailwindCSS + shadcn/ui
- **Backend:** Node.js + tRPC + Drizzle ORM
- **AI:** Python + FastAPI
- **Database:** PostgreSQL + Redis + MinIO
- **Infra:** Docker Compose + Nginx + Turborepo

## Quick Start

```bash
pnpm install
pnpm dev
pnpm build
```
